# 🧵 Sistema de Controle de Corte

Sistema web desenvolvido para melhorar o controle de cortes na produção, permitindo acompanhar o andamento dos trabalhos entre turnos de forma organizada e confiável.

---

## 📌 Problema

Dificuldade em:
- Saber onde cada corte parou
- Garantir continuidade entre turnos
- Evitar erros de planilhas manuais
- Ter visibilidade dos cortes em andamento

---

## 💡 Solução

Sistema que permite:
- Cadastro de cortes (número, produto, mesa)
- Registro de produção por turno
- Continuidade automática a partir da última folha
- Histórico completo por corte
- Visualização de cortes em andamento

---

## ⚙️ Funcionalidades

- 🔍 Buscar corte
- ➕ Criar novo corte
- 🔄 Continuidade entre turnos
- 📊 Histórico de produção
- 📋 Lista de cortes em andamento

---

## 🛠️ Tecnologias

- Node.js + Express
- TypeScript
- SQLite
- HTML, CSS e JavaScript
